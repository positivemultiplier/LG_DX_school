# LG_DX_school

초심을 잃지말자# 🚀 LG DX School 3기 학습 여정

**"초심을 잃지말자"** - 이 저장소는 LG DX School 3기 과정의 모든 학습 내용과 성장 과정을 기록하는 개인적인 공간입니다.

---

## 🧭 이 저장소를 사용하는 방법 (How to Use)

이곳을 여러분의 학습 베이스캠프로 만드세요!

1.  **학습 내용 정리**: 각 과목 폴더(`01_Foundation` 등)의 `README.md`를 수정하거나, 새로운 마크다운 파일을 만들어 배운 내용을 기록하세요.
2.  **코드 실습**: Python, SQL 등 실습 코드는 해당 과목 폴더 내에 저장하여 관리하세요.
3.  **개인 자료 추가**: 강의 외에 찾은 유용한 아티클, 논문, 스터디 노트는 `resources` 폴더에 자유롭게 추가하세요.
4.  **프로젝트 관리**: 미니 프로젝트와 최종 프로젝트의 모든 산출물은 `projects` 폴더에서 체계적으로 관리하세요.

---

## 🗺️ 전체 학습 로드맵 (Learning Roadmap)

우리의 학습 여정은 기초 역량 강화부터 시작하여, DX 핵심 방법론을 거쳐 최종 실무 프로젝트로 완성됩니다.

*전체 흐름을 보려면 [학습 로드맵(`learning_roadmap.md`)](./learning_roadmap.md) 파일을 확인하세요.*
