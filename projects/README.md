# 🏆 Phase 3: Projects (프로젝트 결과물)

이 폴더는 LG DX School 과정에서 배운 모든 지식과 기술을 실제 문제에 적용하고 해결하는 프로젝트 산출물을 관리하는 공간입니다. 이론을 넘어 실전 경험을 통해 DX 전문가로 성장하는 마지막 단계입니다.

---

## 🎯 이 단계의 학습 목표

- **문제 해결 능력**: 실제 비즈니스 데이터를 바탕으로 문제를 정의하고, 데이터 기반의 해결책을 도출합니다.
- **실무 경험**: 현업 전문가와 협업하며 실무 프로젝트 수행 능력을 강화합니다.
- **포트폴리오 구축**: DX 역량을 증명할 수 있는 구체적이고 설득력 있는 프로젝트 결과물을 완성합니다.

---
