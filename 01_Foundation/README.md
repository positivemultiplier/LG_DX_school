# 📚 Phase 1: Foundation (기초 역량 강화)

본격적인 DX 학습에 앞서, 데이터 전문가로서 필요한 핵심 기초 역량을 다지는 공간입니다. 이 단계에서는 Python 프로그래밍부터 데이터 핸들링, 기초 통계까지 데이터 분석의 전 과정에 대한 튼튼한 기반을 구축합니다.

---

## 🎯 이 단계의 학습 목표

- Python을 활용하여 데이터를 자유롭게 다룰 수 있는 프로그래밍 능력을 갖춥니다.
- 데이터 정제 및 전처리의 중요성을 이해하고 실무 기술을 익힙니다.
- 데이터 분석의 근간이 되는 통계학적 기본 개념을 명확히 이해합니다.
